 SQL Data Analysis Internship – Task 1
 Objective--
Build a Student Management Database and analyze student performance using SQL.

## Tools Used
- MySQL Workbench
- SQL

## Dataset
Student data containing:
- Student ID
- Name
- Age
- Gender
- Department
- Marks

## Queries Performed
1. Display all students.
2. Calculate average marks.
3. Find highest marks scorer.
4. Department-wise average marks.
5. Students scoring above 80 marks.
6. Count total students.
